Web3 and blockchain!

With the emergence of Web3, the knowledge of decentralized application is becoming a mandatory requirement.

Which is why I have created a Calendar appointment app using Web3. This project has a frontend integrated with Solidity, which also make use of a backend (tested using Insomnia app). The project allows a person to schedule an appointment with the owner of the calendar, paying him/her Ethereum based on a rate set by the owner.

The appointment scheduler will also receive an SMS text message in their phone which provides a link to details of the transaction.

This project uses

Solidity, Javascript, Metamask, Hardhat, React, Webhooks, Material UI, Insomnia and Ngrok
